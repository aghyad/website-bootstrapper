== Website Bootstrapper

This bare-bones app is meant to be used as is, to create a quick and functional structure of a website on ruby 2.1.1 and rails 4.0.3

Please clone the repository at your own risk and make sure to write tests and customize the config/database.yml and config/config.yml files.

make sure also to run bundle install, rake db: create:all, rake db:migrate

You can run your rspec test using the command 'rspec'

That's all. Happy Coding!
